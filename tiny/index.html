This is a tiny internet

Documents on the web are written in a language called HTML, which stands for hypertext markup language.

Text on a website can contain links to other documents. That's what makes the web more useful than a regular document. Recently, people have decided that it's a good idea to allow program code to reside in html documents too. Whether this is a good idea or not is still unclear.
